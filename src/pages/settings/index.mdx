# Account settings

These endpoints allow you to retrieve, update your Account settings.

## Retrieve account pixels

<Row>
  <Col>
    <Properties>
      <Property name="facebook_pixels" type="[Pixel!]!">
				The account facebook_pixels.
				```
				type Pixel {
					label: String!
					value: String!
				}
				```
			</Property>
			<Property name="snapchat_pixels" type="[Pixel!]!">
				The account snapchat_pixels.
				```
				type Pixel {
					label: String!
					value: String!
				}
				```
			</Property>
			<Property name="tiktok_pixels" type="[Pixel!]!">
				The account tiktok_pixels.
				```
				type Pixel {
					label: String!
					value: String!
				}
				```
			</Property>
    </Properties> 
  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="https://services.lightfunnels.com/api/v2">
    ```
    query AccountQuery{
      account {
        facebook_pixels {
          label
          value
        }
        snapchat_pixels {
          label
          value
        }
        tiktok_pixels {
          label
          value
        }
      }
    }
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {
        "account": {
          "facebook_pixels": [
            {
              "label": "My fb pixel",
              "value": "Some pixel value"
            }
          ],
          ...
        }
      }
    }
    ```
  </Col>
</Row>

## Retrieve Integrations

<Row>
  <Col>
    <Properties>
      <Property name="integrations" type="[Integration!]!">
				Account integrations.
        ```
        type Integration implements Node {
          id: ID!
          platform: String!
          label: String!
          details: IntegrationDetails
        }

        scalar IntegrationDetails
        ```
			</Property>
    </Properties> 
  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="https://services.lightfunnels.com/api/v2">
    ```
    query AccountQuery{
      account {
        integrations {
          id
          label
          platform
          details
        }
      }
    }
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "data": {
        "account": {
          "integrations": [
            {
              "id": "integration id",
              "label": "testcapi",
              "platform": "facebook"
              "details": {
                "pixels": ["some pixel id"]
              },
            }
          ]
          ...
        }
      }
    }
    ```
  </Col>
</Row>
